# 專案重構改寫計劃需求

## 1. 專案背景
本專案旨在對現有的軟體系統進行重構與改寫，以提升系統的可維護性、可擴展性及性能。
未來要將專案開源，正式專案名稱定為 “templexl” 一個基於openpyxl的Excel模板引擎，專案將提供一個簡單易用的API，允許用戶通過模板生成Excel文件，並支持多種數據源和格式，不需要安裝EXCEL也能使用。

## 2. 專案目標
1. 正式專案名稱改為 “templexl”，將原始程式調整命名 `src/excel_template_renderer`

2. 檢視評估現有原始程式碼，確認其功能、架構及可維護性，並提出改進建議。
  - 檢視現有程式是否能夠完全適用於python 3.12 以上版本
  - 渲染大量數據（以ＥＸＣＥＬ文件最大行數為基準），要能夠在合理時間內完成渲染，並且不會出現記憶體溢位或性能瓶頸。
  - 符合現代資訊安全標準，避免使用過時或不安全的函式庫。
  - 分析現有專案程式，提出重構與改寫的建議，並評估其可行性。
